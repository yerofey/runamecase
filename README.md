# runamecase
A package to convert russian names from nominative to any other case.

## Install
```
$ npm i runamecase
```
OR
```
$ yarn add runamecase
```

## Usage
```js
const runamecase = require('runamecase');

runamecase('Ерофей', 3); // => 'Ерофея'
```
